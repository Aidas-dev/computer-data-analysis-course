# Technology Stack

## Language
- **Python 3.14+** - Latest Python version for optimal performance and modern features

## Core Data Manipulation
- **pandas (>=2.0,<3.0)** - Data structures and data analysis tools
- **numpy (>=1.24,<3.0)** - Numerical computing library

## Data Visualization
- **matplotlib (>=3.7,<4.0)** - Comprehensive plotting library
- **seaborn (>=0.12,<1.0)** - Statistical data visualization based on matplotlib

## Statistical Analysis
- **statsmodels (>=0.14,<1.0)** - Statistical models, hypothesis testing, and regression analysis
- **pyfixest** - Fast high-dimensional fixed effects regression

## Exploratory Data Analysis
- **ydata-profiling (>=4.0,<5.0)** - Automated exploratory data analysis with profile reports

## Econometrics
- **marginaleffects** - Compute and plot marginal effects for regression models

## File Format Support
- **xlrd (>=2.0,<3.0)** - Read legacy Excel (.xls) files
