# Product Definition

## Project Overview
A personal learning repository for the Computer Data Analysis course at VU QE, designed to support self-directed study and replication of econometric research papers.

## Target Users
- **Primary:** QE students (self-learners) using the repository for personal study and skill development
- **Secondary:** Course instructors who may reference the materials

## Learning Objectives
1. **Research Replication:** Replicate academic papers (e.g., Sala-i-Martin's "I Just Ran Two Million Regressions") to understand econometric methods
2. **Statistical Analysis:** Master regression analysis, hypothesis testing, and statistical modeling techniques using Python
3. **Data Processing:** Learn to clean, transform, and explore real-world datasets (IRS, MIP, etc.)

## Key Features
- **Jupyter Notebooks:** Interactive notebooks combining code, analysis, visualizations, and explanatory text
- **Task-Based Structure:** Organized by course tasks (Task 1, Task 2, Lesson 2) with dedicated data directories
- **Explicit Instructions:** Each exercise follows the provided instructions from the data folder tutorials
- **Automated EDA:** Leverage ydata-profiling for rapid exploratory data analysis reports

## Technical Constraints
- **Python 3.14+:** Use the latest Python 3.14+ for optimal performance and features
- **Version-Controlled Dependencies:** All packages pinned with version ranges in requirements.txt for reproducibility
- **Git Version Control:** Track all changes with structured commits and clear history

## Success Criteria
- All course tasks completed with working notebooks
- Reproducible analysis pipelines that can be re-run with consistent results
- Clear documentation of methods and findings in each notebook
